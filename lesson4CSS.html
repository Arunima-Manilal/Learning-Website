CSS Selectors

CSS selectors are used to "find" or select the HTML elements you want to style.

We can divide CSS selectors into five categories:

  • Simple selectors (select by name, id, class)
  • Combinator selectors (based on relationships)
  • Pseudo-class selectors (based on state)
  • Pseudo-element selectors (part of an element)
  • Attribute selectors (based on attribute values)

The Element Selector

Selects elements by their name.

p {
  text-align: center;
  color: red;
}

The ID Selector

Uses the id attribute to select a unique element.

#para1 {
  text-align: center;
  color: red;
}

Note: ID names cannot start with a number!

The Class Selector

Uses the class attribute to select elements.

.center {
  text-align: center;
  color: red;
}

To target a specific element type with a class:

p.center {
  text-align: center;
  color: red;
}

You can apply multiple classes to an element:

<p class="center large">This paragraph refers to two classes.</p>

The Universal Selector

Selects all elements on the page.

* {
  text-align: center;
  color: blue;
}

The Grouping Selector

Groups multiple selectors with the same styles.

h1, h2, p {
  text-align: center;
  color: red;
}

All CSS Simple Selectors

Selector Example Description
#id #firstname Selects the element with id="firstname"
.class .intro Selects all elements with class="intro"
* * Selects all elements
element p Selects all <p> elements
element, element div, p Selects all <div> and all <p> elements
