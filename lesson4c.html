Lesson 4: C Statements

What Are Statements?

A computer program is a list of instructions to be executed by a computer. In C, these instructions are called statements.

The following is a simple statement that tells the computer to print something on the screen:

Example:

printf("Hello World!");

Important: Always end a statement with a semicolon ;. If you forget it, your program will throw an error:

Example (missing semicolon):

printf("Hello World!")
error: expected ';' before 'return'

Multiple Statements

Most C programs contain many statements. These are executed one by one in the order they appear:

Example:

printf("Hello World!");
printf("Have a good day!");
return 0;

Example Explained

  • printf("Hello World!"); → prints "Hello World!"
  • printf("Have a good day!"); → prints "Have a good day!"
  • return 0; → ends the program successfully

For now, just remember: every C statement must end with a semicolon to avoid errors.
