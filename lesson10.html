Lesson 4: HTML class Attribute

What is the class Attribute?

The HTML class attribute is used to specify a class for an HTML element. Multiple elements can share the same class.

Using Classes with CSS

The class attribute can link to a style defined in a stylesheet.

<style>
.city {
  background-color: tomato;
  color: white;
  border: 2px solid black;
  margin: 20px;
  padding: 20px;
}
</style>

<div class="city">
  <h2>London</h2>
  <p>London is the capital of England.</p>
</div>

More Examples

<span class="note">Important</span>

<style>
.note {
  font-size: 120%;
  color: red;
}
</style>

This will style both span elements the same way.

Syntax for Class

To create a class in CSS, use a period (.) followed by the class name:

.city {
  background-color: tomato;
  color: white;
  padding: 10px;
}

Apply it to elements like:

<h2 class="city">London</h2>

Multiple Classes

You can assign more than one class to an element, separated by spaces:

<h2 class="city main">London</h2>

Different Elements Sharing the Same Class

Different elements can share a class:

<h2 class="city">Paris</h2>
<p class="city">Paris is the capital of France</p>

JavaScript and Class

JavaScript can select elements by class name using getElementsByClassName():

<script>
function myFunction() {
  var x = document.getElementsByClassName("city");
  for (var i = 0; i < x.length; i++) {
    x[i].style.display = "none";
  }
}
</script>

Note: You’ll learn more about JavaScript in a later chapter!

Chapter Summary

  • The class attribute defines one or more class names.
  • CSS and JavaScript can both use classes to select and style elements.
  • Multiple elements can share the same class name.
  • Class names are case sensitive.
