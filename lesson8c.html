Lesson 7: C Comments

Why Use Comments?

Comments are used to explain code, improve readability, or prevent code from executing. They can be single-line or multi-line comments.

Single-line Comments

Start single-line comments with two slashes //. Everything after // on the same line will be ignored by the compiler.

Example (Before a Line):

// This is a comment
printf("Hello World!");

Example (End of a Line):

printf("Hello World!"); // This is a comment

Multi-line Comments

Start with /* and end with */. Everything between is ignored by the compiler.

Example:

/* The code below will print
Hello World! to the screen */
printf("Hello World!");

Which Type Should You Use?

Use // for short explanations and /* */ for longer comments or when commenting out blocks of code.

Note: Before C99 (released in 1999), only multi-line comments were allowed in C.
