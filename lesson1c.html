Lesson 1: C Introduction

What is C?

C is a general-purpose programming language created by Dennis Ritchie at the Bell Laboratories in 1972.

  • It is a fundamental language in the field of computer science.
  • It is strongly associated with UNIX, as it was used to develop the UNIX operating system.
  • Despite its age, it remains highly popular and relevant.

Why Learn C?

  • One of the most popular programming languages in the world
  • Knowing C makes learning other languages like Java, Python, C++, and C# easier
  • C is very fast compared to Java and Python
  • Versatile and suitable for applications and low-level technologies

Difference between C and C++

  • C++ was developed as an extension of C
  • Both have similar syntax
  • Main difference: C++ supports object-oriented programming (classes and objects), while C does not

Get Started

This tutorial will teach you the basics of C programming. No prior experience is required!

By the end of this course, you’ll be able to write basic C programs and create real-life examples.

Hello World in C

#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

Explanation:

  • #include <stdio.h> includes the standard input/output library
  • int main() is the starting point of the program
  • printf is used to print output to the screen
  • return 0; indicates that the program ended successfully
