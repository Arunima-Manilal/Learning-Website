CSS Comments

CSS comments are used to explain your code and make it easier to read and maintain. They are not displayed in the browser.

CSS comments start with /* and end with */. Browsers ignore comments.

Single-line Comment

/* This is a single-line comment */
p {
  color: red;
}

Comment Within a Declaration

p {
  color: red;  /* Set text color to red */
}

Comment in the Middle of a Line

p {
  color: /*red*/blue;
}

Multi-line Comment

/* This is
a multi-line
comment */

p {
  color: red;
}

Combining HTML and CSS Comments

You can also add comments in HTML using <!-- ... -->.

<!DOCTYPE html>
<html>
<head>
<style>
p {
  color: red; /* Set text color to red */
}
</style>
</head>
<body>

<h2>My Heading</h2>

<!-- These paragraphs will be red -->
<p>Hello World!</p>
<p>This paragraph is styled with CSS.</p>
<p>HTML and CSS comments are not shown in the output.</p>

</body>
</html>
