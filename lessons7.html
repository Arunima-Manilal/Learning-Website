Lesson 6: HTML Images

The <img> Tag

HTML images are defined with the <img> tag. This tag is empty, meaning it contains attributes only and does not have a closing tag.

Basic Image Example

<img src="img_girl.jpg" alt="Girl in a jacket">

Image Attributes

  • src - The path to the image
  • alt - Alternate text if the image can‚Äôt be displayed
  • width and height - Image size in pixels
<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">

Absolute vs. Relative URLs

  • Absolute URL - Includes the full web address:
    • <img src="https://example.com/image.jpg">
  • Relative URL - Refers to a file within your website:
    • <img src="images/image.jpg">

The alt Attribute

Use alt to describe the image if it can't load or for screen readers:

<img src="img_typo.jpg" alt="Girl with a jacket">
