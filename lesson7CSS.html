CSS Backgrounds

The CSS background properties are used to add background effects for elements. You'll learn:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • background (shorthand)

background-color

This property sets the background color of an element.

body {
  background-color: lightblue;
}

Color values can be set using:

  • Color name – red
  • HEX – #ff0000
  • RGB – rgb(255, 0, 0)
h1 background: green
div background: lightblue
p background: yellow

Opacity / Transparency

The opacity property sets how transparent an element is (0.0 to 1.0):

div {
  background-color: green;
  opacity: 0.3;
}
opacity: 1 (fully opaque)
opacity: 0.6
opacity: 0.3
opacity: 0.1

Note: Child elements will also become transparent when using the opacity property.

Transparency using RGBA

To apply transparency to the background only (not the text), use rgba() instead of opacity:

div {
  background: rgba(0, 128, 0, 0.3);
}
100% opacity
60% opacity
30% opacity
10% opacity

RGBA adds an alpha channel to RGB for transparency. Format: rgba(red, green, blue, alpha).
