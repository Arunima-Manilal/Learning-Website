Lesson 2: HTML Attributes

What are HTML Attributes?

  • Provide additional information about elements
  • Always appear in the start tag
  • Usually written as name="value"

The href Attribute

Used in anchor tags to define link destination:

<a href="https://www.w3schools.com">Visit W3Schools</a>

The src Attribute

Specifies image location in the <img> tag:

<img src="img_girl.jpg">

Two ways to use src:

  • Absolute URL: src="https://www.example.com/image.jpg"
  • Relative URL: src="img/image.jpg"

Width and Height Attributes

Used to control image size:

<img src="img_girl.jpg" width="500" height="600">

The alt Attribute

Specifies alternate text for images:

<img src="img_girl.jpg" alt="Girl with a jacket">

Example with a missing image:

<img src="img_typo.jpg" alt="Girl with a jacket">

The style Attribute

Adds inline styles to elements:

<p style="color:red;">This is a red paragraph.</p>

The lang Attribute

Used in the <html> tag to specify page language:

<html lang="en">

Can include country code:

<html lang="en-US">

The title Attribute

Displays a tooltip when hovering over an element:

<p title="I'm a tooltip">This is a paragraph.</p>

Best Practices

  • Use lowercase for attribute names
  • Quote attribute values
  • Use double quotes by default
  • Use single quotes if value contains double quotes

Examples

<a href="https://www.w3schools.com/html/">Visit our HTML tutorial</a>
<p title='John "ShotGun" Nelson'>Nickname example</p>

Chapter Summary

  • All elements can have attributes
  • href for links
  • src, width, height, alt for images
  • style for inline styling
  • lang for language
  • title for tooltips
