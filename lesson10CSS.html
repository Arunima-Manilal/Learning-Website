CSS Height, Width and Max-width

The height and width properties are used to define the height and width of an element.

The max-width property sets the maximum width an element can take.

Setting Height and Width

These properties set the content area of an element (excluding padding, border, and margin).

div {
  height: 200px;
  width: 50%;
  background-color: powderblue;
}
height: 200px, width: 50% 
div {
  height: 100px;
  width: 500px;
  background-color: powderblue;
}
height: 100px, width: 500px
Note: Height and width apply only to the content box. Padding, borders, and margins are extra!

Setting Max-width

Using max-width allows the element to scale down on smaller screens without overflowing.

div {
  max-width: 500px;
  height: 100px;
  background-color: powderblue;
}
height: 100px, max-width: 500px
Tip: Try resizing the browser window. The max-width box will shrink, while a fixed width box may overflow and create scrollbars.
