CSS Syntax

A CSS rule consists of a selector and a declaration block.

CSS Selector

The selector points to the HTML element you want to style.

Declaration Block

The declaration block contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon.

Declaration blocks are surrounded by curly braces { }.

Multiple declarations are separated by semicolons ;.

Example

This example will style all <p> elements to have red text and be center-aligned:

p {
  color: red;
  text-align: center;
}
