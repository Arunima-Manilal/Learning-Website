What is Python?

Python is a popular programming language. It was created by Guido van Rossum and released in 1991.

It is used for:

  • Web development (server-side)
  • Software development
  • Mathematics
  • System scripting

What can Python do?

  • Create web applications on a server
  • Workflows in software
  • Connect to and modify databases
  • Handle big data and perform complex math
  • Rapid prototyping and production software

Why Python?

  • Cross-platform (Windows, Mac, Linux, Raspberry Pi)
  • Simple English-like syntax
  • Fewer lines of code needed
  • Interpreter-based execution
  • Supports procedural, object-oriented, and functional programming

Good to Know

The most recent major version of Python is Python 3. We'll use it in this tutorial.

Python can be written in a text editor or an IDE like Thonny, PyCharm, NetBeans, or Eclipse.

Python Syntax vs Other Languages

  • Designed for readability, influenced by math and English
  • Uses new lines to end commands (not semicolons)
  • Uses indentation (not curly braces) to define scope
