Lesson 5: C Output (Print Text)

Printing Text with printf()

To output values or print text in C, you can use the printf() function.

Example:

#include <stdio.h>

int main() {
  printf("Hello World!");
  return 0;
}

Double Quotes

When working with text, it must be wrapped in double quotes "".

Correct:

printf("This sentence will work!");

Incorrect:

printf(This sentence will produce an error.);

If you forget the quotes, you’ll get a compilation error!

Using Multiple printf() Statements

You can use as many printf() functions as needed. But note: it doesn't automatically add a new line between outputs.

Example:

#include <stdio.h>

int main() {
  printf("Hello World!");
  printf("I am learning C.");
  printf("And it is awesome!");
  return 0;
}

This code will output:
Hello World!I am learning C.And it is awesome!

We'll learn how to add line breaks using \n in the next lesson!
