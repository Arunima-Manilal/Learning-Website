Lesson 5: HTML Links

What Are HTML Links?

Links allow users to navigate to other web pages or external websites. They are created using the <a> tag.

The <a> Tag

The <a> tag (anchor tag) is used to create hyperlinks. The destination URL is set in the href attribute.

<a href="https://www.w3schools.com">Visit W3Schools</a>

Opening Links in a New Tab

Use target="_blank" to open the link in a new browser tab:

<a href="https://www.google.com" target="_blank">Open Google in a new tab</a>

Linking to Pages in Your Website

You can also link to another page in your own website:

<a href="about.html">Go to About Page</a>

Link Attributes

  • href - Specifies the destination address
  • target - Defines where to open the link
  • title - Tooltip text on hover

Styled Links

You can style links using the style attribute or CSS:

<a href="#" style="color:yellow;">Yellow Link</a>
