Lesson 4: HTML Paragraphs

What Are HTML Paragraphs?

Paragraphs are blocks of text that are separated from other blocks. They help break content into readable chunks.

The <p> Tag

HTML paragraphs are defined using the <p> tag:

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

Paragraph Formatting

  • Browsers automatically add a blank line (margin) before and after each paragraph
  • Use multiple <p> tags instead of line breaks for proper formatting

Don't Use Multiple <br> Tags

While you can use <br> to break lines, don't use it excessively to create spacing between paragraphs. Use proper tags like <p> for better structure.

<p>This is correct.</p>

<br><br>
<!-- Not recommended for spacing! -->

Using the style Attribute

You can use the style attribute to change how a paragraph looks. For example, make text red:

<p style="color:red;">This is a red paragraph.</p>

Quiz Question

  • <h1>
  • <p>
  • <div>

Tag Reference

  • <p> - Defines a paragraph
  • <br> - Inserts a line break
