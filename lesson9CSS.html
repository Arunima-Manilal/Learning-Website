CSS Background Image

The background-image property is used to set an image as the background of an element.

Basic Usage

By default, the image will repeat both vertically and horizontally to cover the element completely.

body {
  background-image: url("paper.gif");
}
This is a sample with paper.gif as the background image.

Readability Warning

If you choose a background image that is too detailed or bright, it may reduce the readability of your text.

body {
  background-image: url("bgdesert.jpg");
}
This is a sample with bgdesert.jpg. It may make text harder to read.
Note: Choose background images that do not distract from or clash with the text content.

Applying to Specific Elements

You can apply background images to specific elements like a paragraph or a div:

p {
  background-image: url("paper.gif");
}
