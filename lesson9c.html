Lesson 8: C Variables

Variables are containers for storing data values like numbers or characters.

Types of Variables

  • int - stores whole numbers like 123 or -123
  • float - stores numbers with decimals like 19.99
  • char - stores single characters like 'a' or 'B'

Declaring Variables

To create a variable, use the format:

type variableName = value;

Example:

int myNum = 15;

Declare First, Assign Later

// Declare
int myNum;

// Assign
myNum = 15;

Output Variables

Using printf() for text works like this:

printf("Hello World!");

But printing a variable directly like this won't work:

int myNum = 15;
printf(myNum); // ❌ This won't work!

To print variables, you need something called format specifiers — which we’ll explore in the next lesson.
