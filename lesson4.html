Lesson 3: HTML Headings

What Are HTML Headings?

HTML headings are titles or subtitles that help organize content on your webpage.

HTML Heading Tags

Headings are defined with the <h1> to <h6> tags:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Importance of Headings

  • <h1> is the most important heading
  • <h6> is the least important
  • Search engines use headings to understand content structure
  • Users skim pages by reading headings first
  • Only use headings for actual headings – not just for making text big

Default Sizes and Styling

Each heading has a default font size set by the browser. You can customize the size with the style attribute:

<h1 style="font-size:60px;">Heading 1</h1>

Browser Behavior

Browsers add default margin (space) before and after heading elements to make them stand out.

Tag Reference

  • <html> - Defines the root of an HTML document
  • <body> - Defines the content/body of the document
  • <h1> to <h6> - Defines HTML headings
