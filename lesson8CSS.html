CSS Backgrounds

The CSS background properties are used to add background effects for elements.

Key background properties:

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • background (shorthand)

Background Color

Set a background color using a name, HEX, or RGB value:

body {
  background-color: lightblue;
}

You can also apply background colors to individual elements:

h1 {
  background-color: green;
}

div {
  background-color: lightblue;
}

p {
  background-color: yellow;
}
This is an h1 element background: green
This is a div background: lightblue
This is a p background: yellow

Opacity / Transparency

The opacity property affects the transparency of the whole element (including text and children):

div {
  background-color: green;
  opacity: 0.3;
}
Opacity: 1 (fully opaque)
Opacity: 0.6
Opacity: 0.3
Opacity: 0.1

Note: Using opacity affects all child content, which may make text hard to read.

Transparency using RGBA

Use rgba() to apply transparency to the background only, while keeping text fully visible:

div {
  background: rgba(0, 128, 0, 0.3); /* Green with 30% opacity */
}

RGBA format: rgba(red, green, blue, alpha) where alpha is from 0.0 to 1.0.
