Lesson 2: Get Started With C

Requirements

To start using C, you need two things:

  • A text editor (like Notepad) to write your C code
  • A compiler (like GCC) to translate your code into something the computer can understand

We will use an IDE that combines both tools into one environment.

Installing an IDE

An IDE (Integrated Development Environment) allows you to write, compile, and debug code in one place.

Popular IDEs for C include:

  • Code::Blocks
  • Eclipse
  • Visual Studio

We recommend Code::Blocks for beginners. It's easy to use and completely free.

Download it from: codeblocks.org and choose the mingw-setup.exe version to get both the editor and the compiler.

Your First C Program

  1. Open Code::Blocks
  2. Go to File > New > Empty File
  3. Write the following code and save it as myfirstprogram.c

Code:

#include <stdio.h>

int main() {
  printf("Hello World!");
  return 0;
}

Run Your Code

  1. Click Build > Build and Run
  2. The output should look like:
Hello World!
Process returned 0 (0x0) execution time : 0.011 s
Press any key to continue.
