Lesson 3: C Syntax

Understanding Basic Syntax

Let's break down the most basic C program line by line:

Example:

#include <stdio.h>

int main() {
  printf("Hello World!");
  return 0;
}

Explanation:

  • Line 1: #include <stdio.h> includes the Standard Input Output library, which lets us use functions like printf().
  • Line 2: A blank line. C ignores whitespace, but it helps improve readability.
  • Line 3: int main() is the main function from where execution starts. The code inside { } gets executed.
  • Line 4: printf("Hello World!"); prints the text "Hello World!" to the screen.
  • Note: Every C statement ends with a ; (semicolon).
  • Line 5: return 0; ends the main() function and returns 0 to signal successful execution.
  • Line 6: The closing curly bracket } marks the end of the main() function.

Compact Version:

int main(){printf("Hello World!");return 0;}

This version does the same thing, but isn't as readable. Good formatting makes code easier to maintain!
